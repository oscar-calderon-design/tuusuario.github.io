Que es un programa: conjunto de instruciones escritos en un lemguaje de programacion que le indica a un computador
que tareas especificas realizar 

Que es una variable: Una variable es como una caja con un nombre, donde puedes guardar información numerica o de texto 
E.J: 
nombre = "Oscar"
edad = 20
altura = 1.75

print("Hola", nombre, "tienes", edad, "años y mides", altura, "metros")

Tipos comunes de variables:

Enteros (int): números sin decimales → 5, -10

Flotantes (float): números con decimales → 3.14, 1.75

Cadenas (str): texto → "Hola", "ChatGPT"

Booleanos (bool): valores lógicos → True o False

Que es un lenguaje informal: forma no tecnica ni estricta que se usa para explicar ideas,  instrucciones o conseptos.
E.J :

nombre = input("¿Cuál es tu nombre? ")
print("Hola", nombre)
 

Que es un lenguaje formal: sistema de comunicacion con reglas o especificaciones precisas que se usa para dar instrucciones 

E.J: 

num1 = int(input("Ingresa el primer número: "))
num2 = int(input("Ingresa el segundo número: "))
resultado = num1 + num2
print("La suma es:", resultado)

conclucion:  
 
De este tema aprendí que en la informática todo tiene una forma lógica y estructurada. Comprendí que las **variables** son como pequeños contenedores donde se guarda información que puede cambiar, lo que permite que los programas sean dinámicos y útiles.

También entendí la diferencia entre el "lenguaje informal" y el "lenguaje formal". El primero se usa para explicar ideas con palabras comunes, mientras que el segundo es el que entiende la computadora , con reglas muy precisas. Esto me ayudó a ver que antes de programar, es importante planear lo que se quiere hacer y expresarlo en lenguaje informal o pseudocódigo, para luego transformarlo en código real.

En conclusión, aprendí que la programación no solo se trata de escribir código, sino también de **pensar y comunicar las ideas de manera clara y ordenada,
usando el tipo de lenguaje adecuado para cada etapa del proceso.
